# Tenzies

A fun and interactive React-based dice game. Roll 10 dice and try to get all of them to show the same value. Click on individual dice to "hold" their values between rolls. Celebrate with confetti when you win!

## Features

- Roll 10 dice with random values between 1-6.
- Hold dice by clicking on them to prevent re-rolling.
- Automatic win detection when all dice show the same number.
- Confetti animation upon winning.
- Responsive and interactive UI.
